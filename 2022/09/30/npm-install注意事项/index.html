npm install注意事项
2022-09-30 09:29:59 # npm

npm

最近使用npm安装vue的支持的时候发现使用npm install vue-cli之后使用vue命令会找不到此命令，用list查看却又能看到vue已经安装

解决问题

本来以为是哪边安装出问题了，突然想起可能是安装的时候没有加上全局的选项，加上-g则表示全局安装

npm install vue -g

此时使用vue已经可以正常使用

总结

太久不使用npm了，顺便把npm的一些常用命令给总结下

#当前项目安装的所有模块
$npm list

#列出全局安装的模块 带上[–depth 0] 不深入到包的支点 更简洁
$ npm list -g –depth 0

$ npm search <搜索词> [-g]

#卸载当前项目或全局模块
$ npm uninstall [-g]
